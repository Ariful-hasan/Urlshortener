# Url Shortener App 

### Create .env
```bash 
    1.  change .env.example to ".env".

    2.  create database name "laravel".
```

### CLI:
```bash
composer update
php artisan migrate
npm install
```

### System run command:
```bash
php artisan serve
```

### Browse Url
```bash
http://127.0.0.1:8000/
```
